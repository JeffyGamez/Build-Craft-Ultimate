# Build-Craft-Ultimate
Read Me

This Game is called Build Craft Ultimate it is a simple version of Minecraft for mobile and pc it runs on any device with a web browser 
 
#If you Are on Mobile Read this

to get the game working copy the Final Version for Mobile file contense then open your web browser and type data:text/html;base64, then paste the file contense after that text and hit go now to save it click share and navigate until you find add to favorites or bookmark then add it and call it what you want now just click the shortcut and play

#Controls for Mobile

move using the right joystick look around using the left joystick the up and down arrows above the right joystick move your character up and down to break blocks press triangle to place press square to enable flying make sure the fly button is green and has an arrow if you click it you will no longer be flying this is useful for going up and down stairs to fly again just click the fly button once more. if you want to change blocks click the 3 dots button and navigate through the tabs to the block you want then place it around to save your world in inventory click save then download the file if you want to load your world click load in the inventory and then paste the json file you downloaded into the box then click load to reset a world click reset in inventory special items and blocks the sword does cool animations based on if you click square or triangle the pickaxe mines a 3x3 hole if pressing square while holding it for quicker mining if your place tnt then click the flint and steel item and look at the tnt then click square to ignight it notice all tnt in the world will instantly ignight don't place too much of it you might crash the world is 512x512 infinite upwards goes down until bedrock which is unbreakable to keep you from falling into the void
